## ğŸ’» Getting started

### Requirements

- Gatsby CLI
- NodeJS

**Clone the project and access the folder**

```bash
$ git clone https://github.com/leon-webdev/my-website.git && cd my-website
```

**Follow the steps below**

```bash
# Install the dependencies
$ yarn

# Start the client
$ yarn start
```
